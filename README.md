# lampa_trakt_v2

Плагин для [Lampa](https://lampa.mx/) — синхронизация просмотренного с [Trakt.tv](https://trakt.tv/).
Версия 2: переписан с нуля относительно v1, без вмешательства в нативный `Lampa.Favorite`.

**Статус:** в разработке. Сейчас — Phase 1 MVP (пункт меню → страница со списком Watchlist).

## Идея

В отличие от v1 (синхронизировал нативные папки `Lampa.Favorite` с Trakt — вступал в гонку с `cub.red` и не работал на Tizen TV), v2 живёт **рядом** с нативной картотекой Lampa, а не внутри неё:

- Свой пункт в левом меню навигации.
- Свой экран с пятью статусными секциями: Закладка / Смотрю / Просмотрено / Пауза / Брошено.
- Свой кастомный сайдбар выбора папки на детальной карточке.
- Никаких записей в `Lampa.Favorite` — Trakt является единственным источником правды.

Полная спецификация — в [SPEC_v2.md](SPEC_v2.md).

## Зависимости

На текущем этапе плагин работает как **надстройка** над сторонним плагином `trakt_by_lampame`:

- OAuth, device-code-flow, refresh-токены — за `trakt_by_lampame`.
- Scrobble эпизодов на плеере — за `trakt_by_lampame`.
- Токен читается напрямую из `Lampa.Storage.get('trakt_token')`.

Прокси Trakt API: `https://apx.lme.isroot.in/trakt` (тот же, что использует `trakt_by_lampame`). Финальная независимость (свой `client_id` + свой прокси) — отложена до первых рабочих результатов v2.

## Установка

В настройках Lampa подключить плагин по URL `trakt_v2.js`-файла.

Перед использованием — войти в Trakt через настройки плагина `trakt_by_lampame` (TraktTV). Без действительного токена `Trakt v2` покажет заглушку с подсказкой авторизоваться.

## Дорожная карта

- **Phase 1 MVP** — пункт меню, Activity component, единый список Watchlist. _(текущий этап)_
- **Phase 1 расширение** — пять секций по статусам, классификатор, TMDB-обогащение карточек.
- **Phase 2** — write-actions, кастомный сайдбар выбора папки на детальной карточке, Pending Ops.
- **Phase 3** — ресёрч по подмене/отключению нативного сайдбара `Lampa.Favorite`.
- Эпизод-синхронизация с `Lampa.Timeline` — отложено, см. SPEC §«Решения, отложенные на потом».
